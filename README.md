## 🚀 Empezando

Estas instrucciones le ayudarán a obtener una copia del proyecto en funcionamiento en su máquina local para fines de desarrollo y prueba.

### ⚙️ Instalación

Para instalar todas las dependencias necesarias, ejecute el siguiente comando en la raíz del proyecto:

```bash
mvn install
```

### 🤖 Ejecución de Inventario

Para ejecutar el proyecto, simplemente use el siguiente comando:

```bash
mvn spring-boot:run
```

### 🧪 Pruebas

Para ejecutar las pruebas, ejecute:

```bash
mvn test
```
